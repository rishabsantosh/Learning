This is a static template, there is no bundler or bundling involved!

This Sandbox is to for playing with CodeSandBox and Github to see how both work together and how to sync in both directions

First link this sandbox to your github.

When you've time experiment with this and learn how to update changes made here onto the original repository in Github.

Points:

  • When we want to open and edit a GitHub repository in CodeSandBox, open it through CodeSandBox itself.

  • When we want to just open and view the repository, we can also open through GitHub by appending "box" after github and .com in the url.

  • CORRECT, BUT CHECK To save a sandbox on GitHub first export the sandbox to a new GitHub repository. Then open the sandbox again and check if you have commit changes option for you. If not, that means you cannot commit changes from sandbox to GitHub, so delete the sandbox, and import the exported sandbox and save it, this should work.
